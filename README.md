## Installation

Install from the [Chrome Web Store](https://chromewebstore.google.com/detail/tab-manager-pro-save-rest/ieendfnemamipdokaeakaegmjalmhdfp)

## Usage

1. Click the extension icon
2. Name your group and choose a color
3. Click "Save Current Tabs"
4. Restore anytime by clicking the group name
